# Vidres Valls Web

Web corporativa de Vidres Valls construida con Next.js, next-intl y React.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Estructura principal

- `src/`: aplicación, rutas, componentes y traducciones.
- `public/`: imágenes, logo y assets servidos por Next.
- `docs/`: referencias visuales y guía de estilo.

La web usa catalán como idioma principal, con español e inglés como idiomas adicionales.
